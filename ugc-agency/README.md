# ModernFlow UGC Engine

AI-generated UGC-style video ads as a productized service line for ModernFlow AI.

**Stack:** Claude (Creative Director) + Higgsfield MCP (video/image generation) + client's platform (Shopify, GHL, Meta Ads).

---

## Why this fits ModernFlow

ModernFlow already sells done-for-you automation to service businesses (contractors, HVAC, plumbers) and holds GHL + Calendly + CRM infrastructure. UGC-style creative is the missing top-of-funnel piece:

- **Existing clients** run Meta/local ads and suffer creative fatigue — this is an upsell, not a new sales motion.
- **Shopify DTC brands ($100K–$2M/yr)** are the second segment: they already spend on Meta, need ~20 fresh hooks/month to keep ROAS alive, and don't need award-winning creative — they need volume + angles.
- The economics work because generation cost is near-zero; the value sold is the *system* (angle research → scripts → platform-native video), not the MP4s.

## Operating principles

1. **The counter-narrative beats the consensus.** If every competitor promises "instant results," the winning hook is the one that contradicts the category script.
2. **Consistency beats novelty.** One recognizable persona per vertical, reused every campaign. Recognition converts.
3. **Authentic beats polished.** "Shot on iPhone," slightly imperfect, handheld. Perfect lighting is a bug in this format.
4. **Volume demands curation.** AI removes the production bottleneck; creative judgment about which angles convert becomes *more* valuable, not less. Never ship a batch without a human pass.
5. **Sell output, not time.** Deliverables are priced per pack, not per hour.
