# Short-Form Video Scripts (TikTok / Reels / Shorts)

21 scripts, 20–45 seconds each. Record once, post to all three platforms.
Legend: **[SR]** = screen recording, **[TH]** = talking head (optional face),
**OST** = on-screen text. Every video ends with the same CTA unless noted:

> **CTA:** "Link in bio — it's the toolkit, not a course." + OST `modernflowai.com/toolkit`

Batch tips: record SR clips at 1080×1920 (crop n8n canvas to the action), keep
cuts under 2s, captions on everything (80% watch muted), hook must land in the
first 1.5 seconds — say the OST line out loud as the clip opens.

---

## Pillar A — Proof / Demo (the moat: show real files)

**V1 — "Watch AI score a cold email reply"** [SR]
Hook/OST: "This workflow reads my inbox and scores every reply 1 to 10."
Body: n8n canvas → trigger fires on a (sanitized demo) reply → zoom on the LLM
node output: `score: 8, reason: asked about pricing` → auto-reply node sends the
booking link. VO: "8 or higher, it replies with my calendar automatically. I
never touch warm leads until they book."
CTA: "This is 1 of 4 workflows in the kit. Link in bio."

**V2 — "This is what $297 actually gets you"** [SR]
Hook/OST: "Opening every file in the kit in 60 seconds. Judge for yourself."
Body: timelapse folder walk — zips → 01-playbook (scroll an SOP), prompt
templates, both automation folders, runbooks. No music drop, just fast cuts.
VO names each file plainly. Anti-hype delivery is the point.

The exact workflow file is in the kit — you can deploy it for YOUR clients."

- Question form: "Want to see what an actual agency workflow looks like?"
- Number form: "4 workflows run my agency. This is #2."
- Callout form: "If you bought an AI agency course, look away."
