# SOP 03 — CRM + Client Onboarding

**Owner:** `crm-onboarding-agent` · **Skill:** `/client-onboarding` · **System:** GoHighLevel

## Goal

Keep the pipeline clean and turn every Closed Won into a live, delivering client fast —
this is how we protect retention.

## Pipeline stages

New Lead → Contacted → Call Booked → Proposal Sent → Closed Won → Onboarding → Live →
(Churned). Every contact sits in exactly one stage with an accurate next-action + date.

## Onboarding checklist (run on Closed Won)

1. **Welcome** — welcome email + intake form (business info, service area, hours, current
   tools, review profiles, branding). Kickoff-call timing → SOP 04.
2. **Provision by tier** in GHL:
   - *All tiers:* speed-to-lead (instant text/email + missed-call text-back), online
     booking + reminders, review-request automation.
   - *Growth/Elite:* nurture sequences, reporting dashboard.
   - *Elite:* reactivation/winback campaign.
3. **Connect** — phone number, Google Business Profile, calendar, website form.
4. **Test** — run a test lead through every automation.
5. **Go-live + train** — short training video, confirm first results, set check-in cadence.

## Definition of done

Client is Live: tier-appropriate automations tested and running, integrations connected,
training delivered, CRM updated, check-in cadence set.

## Guardrails

- Never delete/bulk-edit CRM records without confirmation.
- Provision only what the tier paid for.
- Verify with a test before telling the client anything is live.
- Draft all client-facing messages for approval first.
