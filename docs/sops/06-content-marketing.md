# SOP 06 — Content + Marketing

**Owner:** `content-marketing-agent` · **Skill:** `/sales-deck` · **Systems:** Gamma, media generation

## Goal

Produce on-brand sales and marketing collateral that helps close deals and build the brand.

## Brand system (do not drift)

- **Colors:** emerald `#10b981` → teal `#14b8a6` → cyan `#06b6d4` on near-black `#0a0f0d`;
  glass cards `rgba(255,255,255,0.04)`.
- **Type:** Outfit (headings), DM Sans (body), JetBrains Mono `// LABELS`.
- **Aesthetic/voice:** dark-mode SaaS "tech noir"; plain-spoken, ROI-focused, written for
  a busy service-business owner.

## What we make

- **Pitch/sales decks** (Gamma): pain → cost of slow follow-up → our system → 4-step how →
  results → pricing → next step.
- **Proposals** tailored to niche + tier.
- **Social/ads:** one pain per hook.
- **Case studies:** problem → what we set up → measurable outcome (defensible).
- **Visual assets:** on-brand images/video via media-generation tools.

## Procedure

1. Confirm asset, audience, and the single takeaway.
2. Draft copy (outline → full), then generate on-brand visuals.
3. Build in the right tool; return a shareable draft link + editable source.
4. Publishing/scheduling via SOP 08 (Comms), draft first.

## Definition of done

An on-brand draft delivered for review, with placeholders for any real metrics/logos clearly marked.

## Guardrails

- Draft, then publish — nothing public without approval.
- No invented testimonials/logos/metrics; mark `[PLACEHOLDER]`.
- Conflicts with the brand system → flag + propose an on-brand alternative.
