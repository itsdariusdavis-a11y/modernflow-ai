# SOP 04 — Booking + Scheduling

**Owner:** `scheduling-agent` · **Skill:** `/book-call` · **Systems:** Google Calendar, Calendly

## Goal

Make booking effortless and never double-book. Booked calls are the business's conversion
event.

## Defaults

- Strategy/sales call: 30 min · client kickoff: 45 min · check-in: 15 min.
- Public booking link: https://calendly.com/ryan-modernflowai/30min
- Host business hours; 15-min buffers; avoid >3 back-to-back calls.
- Reminders 24h + 1h before, with the call goal and a reschedule link.

## Procedure

1. **New call:** confirm purpose, duration, attendees, **timezone**. Find 2–3 open slots,
   propose them, create the event titled "ModernFlow ⟂ {Company} — {Type}" with an agenda
   in the description.
2. **Inbound positive (from outreach):** offer the next 2–3 slots in their timezone — get
   them booked immediately, don't send them away.
3. **Reschedule/cancel:** free the old slot, offer alternatives, update all invitees.
4. Pass booked-call context to SOP 05 so the recap loop is ready.

## Definition of done

Event created with correct time/timezone, attendees, agenda, and reminders — no conflicts.

## Guardrails

- Confirm before creating/editing/deleting any multi-person event.
- Never double-book the host; respect buffers.
- Get the timezone right — it's the #1 cause of no-shows.
