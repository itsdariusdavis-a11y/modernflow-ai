# SOP 09 — Reporting + KPIs

**Owner:** `reporting-agent` · **Skill:** `/weekly-report` · **Systems:** all (read-only)

## Goal

Give the owner an honest, decision-ready picture of the business each week (or on demand).

## KPIs

- **Top of funnel:** leads added, outreach sent, reply rate, calls booked.
- **Sales:** proposals sent, close rate, new MRR, deals per pipeline stage.
- **Delivery:** clients onboarded, automations live, time-to-go-live, open action items.
- **Retention/health:** active clients, churn, at-risk accounts, overdue follow-ups.
- **Cash:** new setup fees + MRR by tier (Starter/Growth/Elite).

## Procedure

1. Set the window (default last 7 days); note data gaps.
2. Pull numbers read-only from each system (Apollo, CRM, Calendar, Fireflies).
3. Write one page: headline numbers, week-over-week change, what's working, what's
   slipping, and the **top 3 recommended actions**, each tagged with the owner specialist.
4. Offer to post the digest internally via SOP 08 (draft first).

## Definition of done

A one-page report with honest numbers, trends, risks, and 3 owner-tagged recommended actions.

## Guardrails

- Read-only — never modify records or send anything.
- Honest about gaps/small samples; never fabricate a number.
- Lead with the decision: what should change this week?
